Http - The network Protocol

HTTP stands for hypertext transfer protocol and basically put, a communication protocol used on the internet with the purpose of
transferring web pages on the world wide web. The This network protocol is setting the rules for how something is being transferred or communicated
between client computers and web servers in the network, which is made possible by sending HTTP requests and receiving HTTP responses.
Thanks to HTTP, we are able to send and receive data between web pages and servers all around the globe
